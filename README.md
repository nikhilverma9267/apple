# Apple Website Landing Page Clone

A simple Apple-inspired landing page built using **HTML** and **CSS**. This project recreates the first page of Apple's website, featuring a clean navigation bar, hero section, dark theme, and responsive design.

## 🚀 Features

- Apple-style navigation bar
- Hero section with Apple logo and WWDC26 branding
- Modern and responsive layout
- Clean UI using only HTML and CSS
- Beginner-friendly project structure

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

├── index.html
├── style.css
└── images/

## 🎯 Purpose

This project was created to practice:

- HTML page structure
- CSS styling and positioning
- Flexbox layout
- Responsive web design
- Real-world website cloning
